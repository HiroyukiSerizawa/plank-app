# Русский (ru-RU)

> ⚠️ **Примечание:** само приложение пока на английском (запасной локали)

## Название приложения (29 / 30)

```
Plank Now — Таймер Планки Кор
```

## Краткое описание (78 / 80)

```
Киберпанк-таймер планки. Старт, держи, побей личный рекорд. Бесплатно, офлайн.
```
